# 1 项目介绍
基于SSM+Vue的茶叶销售系统：后端 SpringBoot、Mybatis-Plus，前端Vue+ElementUI，具体功能如下：
## 管理端
- 茶叶信息管理：增删改查茶叶信息
- 茶叶类型管理：增删改查茶叶类型
- 用户信息管理：增删改查用户信息
- 订单管理：查看订单信息、订单详情
- 个人中心：查看和修改个人信息
## 用户端
- 商品列表：查看茶叶分类和茶叶列表
- 商品详情：查看商品详细信息、商品加购
- 购物车：查看购物车商品、提交订单
- 我的订单：查看订单记录、订单详情、订单支付
- 个人中心：查看和修改个人信息、管理收货地址
